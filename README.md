# Bhanu Prasad Ulichi - Portfolio

Welcome to my personal portfolio website! I am a **Java Backend Developer & AI Engineer** specializing in building scalable backend systems and intelligent applications — from RESTful APIs with Spring Boot to ML-powered solutions with Python.

## Overview

This repository contains the source code for my personal portfolio. It is built as a highly optimized, single-page web application featuring:

*   **Modern Design:** Glassmorphism, ambient lighting, and custom component utilities.
*   **Vibrant User Experience:** Micro-interactions, smooth scroll, typewriter animations, and dynamic visual flair.
*   **Performance Focused:** Written in vanilla HTML, CSS, and JS to keep the footprint small and load times incredibly fast.
*   **Responsive Layout:** Fully tailored to handle desktop and mobile viewports seamlessly.

## Project Structure

*   `index.html`: The core of the application, encompassing the entire structure, styles (`var(--bg)`, `var(--text)`, etc.), and logic for animations, forms, and interactive components.

## Features

*   **Hero Section:** Highlights my role and key areas of expertise. Let's make an impact!
*   **About Me:** A summary of my professional journey, philosophy, and technical stats.
*   **Skills:** A visualization of my technical stack across backend, AI, and DevOps.
*   **Projects:** Detailed case studies of my applications, including Impulse, an intelligent mood recommendation engine.
*   **Contact:** Direct integration with Formspree for easy communication.

## Quick Start
Since this project uses vanilla web technologies, no build tools are necessary to run it.

1. Clone the repository
```bash
git clone https://github.com/bhanu4734/portfolio.git
```
2. Navigate to the directory
```bash
cd portfolio
```
3. Open `index.html` in your favorite web browser.
